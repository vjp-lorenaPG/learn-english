🎮 Learn English Playing

A web-based educational project designed to help children learn English through interactive and gamified quizzes.

📚 Description

This project is an interactive learning tool aimed at young learners.
It uses visual content and game-based questions to reinforce basic English vocabulary and grammar in a fun and engaging way.

The main goal is to make learning feel like playing, improving motivation and retention.

🚀 Features

Interactive quizzes

Gamified learning experience

Simple and child-friendly interface

Vocabulary related to daily activities

🧩 Technologies Used

HTML

CSS

Genially (embedded interactive content)

🎯 Learning Objectives

Practice basic English vocabulary

Understand and use simple questions and answers

Improve comprehension through interaction

Encourage autonomous learning
