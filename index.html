Sobre el proyecto

Quién lo ha creado

Soy Lorena Palomino, estudiante del ciclo formativo de Desarrollo de Aplicaciones Multiplataforma (DAM). Esta web es uno de mis proyectos del primer año: con los conocimientos adquiridos en el curso he diseñado y construido desde cero una herramienta educativa funcional, responsive y pensada para el mundo real.

Por qué existe esta web

Vivo en una población pequeña donde los niños de Primaria tienen un nivel de inglés bajo y pocas oportunidades de refuerzo fuera del aula. Esta web nació para cubrir esa necesidad: dar acceso fácil a los contenidos de cada unidad, de manera divertida en un formato claro que cualquier familia pueda usar para repasar sin necesidad de saber inglés.

Base curricular

Los contenidos están basados en la serie All About Us Now de Oxford University Press, el libro utilizado en el centro. Los títulos y materiales de esta web son de elaboración propia, inspirados en dicha serie.

Tecnologías utilizadas

Construida con HTML5, CSS3 y Bootstrap 5. Las presentaciones de cada unidad se integran desde Canva. Las 84 páginas de contenido se generan automáticamente con un script en Python, lo que permite mantener y actualizar el sitio de forma ágil.
